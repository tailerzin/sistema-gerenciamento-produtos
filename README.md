# 🛒 Sistema de Gerenciamento de Produtos

Sistema desenvolvido em Java com foco em Programação Orientada a Objetos (POO).

## 🚀 Funcionalidades

* Adicionar produtos
* Listar produtos
* Buscar produtos por nome
* Remover produtos por ID

## 🧠 Tecnologias

* Java
* ArrayList
* POO

## 📂 Estrutura

```
src/pjt/
├── Main.java
├── Produto.java
└── ProdutoService.java
```

## ▶️ Como executar

```
javac Main.java
java Main
```
